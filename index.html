    Welcome to Mausam 2.0 ! Mausam 2.0 is a simple web app which uses the One API 2.5 by Openweather.org along with the Geolocation API to actively show the local weather based on the location accuracy. Meanwhile, the searchbar allows you to search for the weather of any other city or place you desire if it is globally recognized.
